# C# Coursework Portfolio

This repository contains a collection of C# programs developed for academic and educational purposes in my working through the TCM Security C# 101 class. 

---

## ⚠️ Educational Disclaimer (Password Cracker Project)

The `instructor-password-cracker` directory contains a standalone program and a basic local web server template designed purely to demonstrate authentication logic weaknesses (such as brute-force or dictionary lookups). 

* **Scope**: This code is strictly contained and configured to interact exclusively with a mocked, disconnected, self-hosted local test site.
* **Purpose**: Developed solely for coursework assignments to learn about cryptography, string parsing, and defensive programming practices.
* **Ethics**: This tool does not target, exploit, or interact with any active, live, or third-party web services. It contains no malicious exploits or real-world targeting parameters.
